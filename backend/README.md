### Getting Started

1. **Start MongoDB**
   - **As a Service**: `net start MongoDB` (Admin PowerShell)
   - **Manual**: `mongod --dbpath="C:\data\db"`

2. **Start Backend**
   ```bash
   cd backend
   npm install
   node app.js
   ```
